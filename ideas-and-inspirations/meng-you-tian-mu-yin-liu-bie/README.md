# 梦游天姥吟留别

Selected-line song idea from 李白《梦游天姥吟留别》.

## Naming Rule

- New selected-line song title: `梦游天姥吟留别`.
- Existing website adapted item: `梦游天姥吟留别 · 旧梦版`.
- Existing website full original-poem item: `梦游天姥吟留别 · 原诗版`.

## Intent

Use the most musical and emotionally decisive lines from the poem instead of
forcing a full recitation. The song should move from ascent and vision to
waking loss, then to freedom:

```text
脚著谢公屐，身登青云梯。
半壁见海日，空中闻天鸡。
惟觉时之枕席，失向来之烟霞。
世间行乐亦如此，古来万事东流水。
别君去兮何时还？且放白鹿青崖间，须行即骑访名山。
安能摧眉折腰事权贵，使我不得开心颜。
```

Keep `青云梯`, `东流水`, and `开心颜` as memorable hooks.
